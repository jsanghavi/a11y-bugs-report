# ASE Accessibility Bugs Report

Interactive HTML report for tracking and analyzing accessibility bugs in the Agentic Setup Experience team.

## Features

- 📊 Summary dashboard with metrics by effort level
- 🔍 Live search and filtering
- ↕️ Sortable columns
- 🎨 Color-coded priorities and effort levels
- 📋/🤖 Data source indicators

## View Report

Open `index.html` in your browser or visit the GitHub Pages link (once deployed).
